opencluster.opencluster.lognegexp_normal(par, x)¶

Calculate f(x) where f is parallax approximation function.

It models the field as the sum of log and negative exp, and the cluster as a gaussian.

Parameters

  • par (array of float) – Must contain in order: k, a, h, r, kc, u, s

  • x (float)

Returns

y – y = f(x) = k.log(x/a) + h.e^(-x/r) + (kc/sqrt(2.pi.s)).e^(-((x-u)^2)/2.s^2)

Return type

float
